Filter Metodu

Elemanları tek tek gezer ve içerisindeki koşul doğruysa bu elemanı geri döner. Koşul yanlışsa elemanı geri dömez arrayden çıkartır.

const points = [70, 75, 25, 35, 10, 80, 27]; const studentPassed = points.filter(point => { return true; });

Burada points ve studentPassed aynidir.

const points = [70, 75, 25, 35, 10, 80, 27]; const studentPassed = points.filter(point => { return false; });

Burada studentPassed boş bir arraydir.

Fillter fonksiyonunun çalışma mantığı budur.

Öğrencilerimizin Puanları

Öğrencilerimizin Puanları 50 den büyükler

const studentPassed = points.filter(point => { return point > 50; });

4 tane objeden oluşan dizim objelerimin içinde 2 tane property var..

const students = [ { name: 'furkan', passed: true }, { name: 'seyda', passed: false }, { name: 'hatice', passed: false }, { name: 'elif', passed: true }, ];

Dersten geçen öğrencilerimi Listelemek istiyorum.

const passingStudent = students.filter(student => { return student.passed; });

Dersten geçemeyen öğrencilerimi Listelemek istiyorum.

const passingStudent = students.filter(student => { return !student.passed; });

Tek satır return üm varsa parantezleri kullanmama gerek yok.

Örnek :

const studentPassed = points.filter(point => point > 50);

const passingStudent = students.filter(student => student.passed);

const passingStudent = students.filter(student => !student.passed);

Map Metodu

Map Metodu Elemanları tek tek dönerken bunları tek başına geriye döndürür. Tek başına geriye döndürmek istiyorsanız.

const points = [70, 75, 25, 35, 10, 80, 27];

const newPoints = points.map(point => { return point + 10; })

İlk başta 70 girecek 80 olacak bütün elemanlar tek tek girecek bütün elemanlar 10 artacak. Bütün elemanlara +10 eklenmiş hali newPoints adlı dizide tutulacak.

Öğrenciler :

const studentsforMap = [ { name: 'furkan', point: 40 }, { name: 'seyda', point: 60 }, { name: 'hatice', point: 30 },

{ name: 'elif', point: 100 } ];

Eğer öğrencinin notu 50 nin aşağısında ise öğretmen bu öğrenciler 15 puan ekleyecek..

const newStudentsPoints = studentsforMap.map(student => {

if (student.point < 50) { return { name: student.name, point: student.point + 15 } }

else{ return student; } });

Reduce Metodu

Reduce Metodu Datayı Oynamamızı Sağlar. Örneğin 50 den büyük 3 tane datam var. Yada daha karmaşık veriler var istediğim veriyi çekmek istiyorsam.Daha doğrusu dandik bir backendciniz varsa datayı modifiye etmek istiyorsanız reduce metodunu kullanabiliriz.

const result = points.reduce((value,point) => { if(point > 50) { value++; } return value; },0); //value değerim 0 oldu.

console.log(result); //Log ==> 3

Öğrenciler :

const studentsforReduce = [ { name: 'furkan', point: 40 }, { name: 'seyda', point: 60 }, { name: 'hatice', point: 30 }, { name: 'elif', point: 100 }, { name: 'furkan', point: 80 }, { name: 'seyda', point: 30 }, { name: 'hatice', point: 60 }, { name: 'elif', point: 40 } ];

Bize böyle bir data gelmiş olsun. Adam verileri name e göre gruplayıp datayı size göndermiyorsa..

Ben Burada Furkan'in aldığı notları bastırmak istiyorum.

const furkanTotal = studentsforReduce.reduce((value,student)=>{ if(student.name == 'furkan'){ value += student.point; } return value; },0);

Burada 1-student.point 40, value değerim başlangıçta 0 40 eklendi value 40 diğer point 80 diğer 40 + 80 = 129

Find Metodu

Id yi bulmada data geldi içerisinde bir değeri bulmada kullanılır. Bir handicap i var, bu örneğin notlarımız var puanlarımız var Burada 70 den yüksek ilk notu getirmek istediğimizde find i kullanabiliriz. Yani içerisindeki koşuş true olduğunda ilk olanı getirir diğerlerine bakmaz bile

const points = [70, 75, 25, 35, 10, 80, 27]; const highPoint = points.find(point => { return point > 70; });

console.log(highPoint); //Log ==> 75
